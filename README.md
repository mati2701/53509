# Descripción
la carpeta es un analizador de una gramática en BNF (foto). detecta errores lexicos y sintacticos, cuando detecta alguno, dice el tipo, marca la línea y 
el espacio en donde se encuentra, si no detecta errores genera por consola, la tabla de lexemas y tokens, genera por consola el árbol de derivación, 
traduce el input en lenguaje JavaScript y lo interpreta.
